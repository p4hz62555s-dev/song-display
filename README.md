# ESP32 LCD Typewriter Display

A simple ESP32 project that creates a typewriter animation on a 16x2 I2C LCD with a blinking LED.

## Features
- Typewriter effect
- LED sync with typing
- 16x2 I2C LCD
- ESP32 compatible
- Great for creative tech reels

## Components
- ESP32 DevKit V1
- 16x2 I2C LCD
- LED
- 220Ω resistor

## Wiring

LCD SDA → GPIO21
LCD SCL → GPIO22
LED → GPIO2
